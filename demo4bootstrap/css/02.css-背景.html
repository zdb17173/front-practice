6. background-size 属性：背景尺寸

background-size: 300px 300px
固定像素，可以使用50% 50%; 100% auto;

background-size: cover
填充满容器 长宽比不变 超出部分隐藏

background-size: contain
完整地显示 长宽比不变 可能留白

7. background-origin 属性：控制背景从什么地方开始显示

background-origin: padding-box
内边距开始显示背景图（默认值）

background-origin: border-box
边框开始显示背景图（虚化部分为边框，盖住背景图）

background-origin: content-box
内容区域开始显示背景图
